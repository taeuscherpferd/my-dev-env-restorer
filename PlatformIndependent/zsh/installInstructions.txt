Run this in current shell to install zsh (assuming using bash on Ubuntu)

--------------------------------------------------------------------------------------
apt install zsh
--------------------------------------------------------------------------------------

Then run

--------------------------------------------------------------------------------------
chsh -s /usr/bin/zsh root
--------------------------------------------------------------------------------------

finally run the following to install oh-my-zsh

--------------------------------------------------------------------------------------
wget https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | zsh
--------------------------------------------------------------------------------------
